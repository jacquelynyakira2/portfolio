# AIM Sound Effects

Classic AOL Instant Messenger sound files.

## Sound Files

| File | Description | When Played |
|------|-------------|-------------|
| `dooropen.mp3` | Door opening sound | When opening a chat window |
| `doorclose.mp3` | Door closing sound | When closing a chat window |
| `IM.mp3` | Message sound | When sending/receiving a message |
| `signon.mp3` | Sign on sound | When you sign into AIM |
| `signoff.mp3` | Sign off sound | When you sign out of AIM |

## Bonus Sounds (included but not currently used)

| File | Description |
|------|-------------|
| `File's Done.mp3` | File transfer complete |
| `You've Got Mail.mp3` | Classic AOL mail notification |
